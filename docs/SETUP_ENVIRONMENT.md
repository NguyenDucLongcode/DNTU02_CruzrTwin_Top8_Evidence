# Setup Environment

This guide helps you set up the environment to run Task 5 and Task 6.

## Steps

1. Install Python 3.10 or higher.
2. Open terminal in the project root folder.
3. Install required packages using pip:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the check environment script:
   ```bash
   python scripts/setup/check_environment.py
   ```

If the script outputs `OVERALL RESULT: PASS`, the environment is ready.
